git命令解析

  • git log --graph --decorate --oneline --all
    • 此命令用于图形化展示git的分支情况
    • --oneline表示用一行显示git的commit信息
    • --decorate会显示commit对应的分支、tag等信息
    • --all表示显示所有分支、可以替换成具体分支
    • --simplify-by-decoration只显示和分支、tag对应的commit
